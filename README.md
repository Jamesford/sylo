# Go Sort Your Labels Out (sylo)

Wasting time setting up lables on GitHub / Gitlab / Gogs?

Automate it.

Sample labels.yml file
```
- name: 'Status: Backlog'
  color: 'F1F8E9'
- name: 'Status: In progress'
  color: 'AED581'
(...)
```

## Install
1. Download binary
2. Run binary

## Build
1. Clone repo
2. `go build`
